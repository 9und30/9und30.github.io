GPU Beschleunigung

Superflüster nutzt standardmäßig die CPU. Mit einer NVIDIA Karte kannst du die Transkription massiv beschleunigen.

Hinweis: Superflüster wechselt automatisch zur CPU, wenn keine CUDA-Treiber gefunden werden. Diese Schritte sind optional.

1. CUDA Toolkit 12.x installieren

Lade das Toolkit von der offiziellen NVIDIA Seite herunter:

NVIDIA CUDA Toolkit Download
  • Wähle: Windows → x86_64 → Deine Version → exe (local)
  • Installiere mit Standard-Einstellungen

2. cuDNN installieren

Erfordert einen NVIDIA Account:

NVIDIA cuDNN Download

# Inhalt des ZIP-Archivs in deinen CUDA-Ordner kopieren:

bin/*.dll → C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.x\bin\

include/*.h → C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.x\include\

lib/x64/*.lib → C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.x\lib\x64\

3. PyTorch mit CUDA Support installieren

pip uninstall torch pip install torch --index-url https://download.pytorch.org/whl/cu121

4. Überprüfung

python -c "import torch; print(torch.cuda.is_available())"
# Sollte "True" ausgeben

5. In Superflüster aktivieren

Öffne die Einstellungen über das Tray-Icon und aktiviere den Schalter "GPU verwenden".
